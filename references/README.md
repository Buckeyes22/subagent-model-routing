# External source policy

This repository contains original routing and prompting guidance derived from publicly available vendor documentation. It links to external publications rather than redistributing complete copies whose reuse terms are not part of this repository's MIT license.

The Claude model references use Anthropic's official system-card sources:

- [Claude Sonnet 5 System Card](https://www.anthropic.com/claude-sonnet-5-system-card)
- [Claude Opus 4.8 System Card](https://www.anthropic.com/claude-opus-4-8-system-card)
- [Claude Fable 5 & Claude Mythos 5 System Card](https://www.anthropic.com/claude-fable-5-mythos-5-system-card)
- [Anthropic model system-card index](https://www.anthropic.com/system-cards)

Vendor names, trademarks, documentation, and system cards remain the property of their respective owners. The repository's MIT license covers this project's original code and documentation; it does not grant rights in externally linked material.

When refreshing a model reference, cite the official hosted source, summarize only what is necessary for the routing guidance, identify project recommendations as evidence-derived, and do not commit a full converted publication without documented redistribution permission.
